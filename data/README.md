# Catálogo de Audio y Referencias Musicales

Esta carpeta está organizada para recibir las fuentes de audio y referencias musicales del proyecto **MusicIA**.

## Estructura de Carpetas

- `audio/influencias/`: Canciones de otros autores para análisis de estilo, estructura, armonía y producción.
- `audio/propias/`: Canciones y composiciones propias del autor para entrenamiento, continuidad de estilo o referencia personal.
- `audio/analisis/`: Archivos generados de análisis (transcripciones MIDI, diagramas de acorde, espectrogramas, extracción de stem, metadatos JSON).

## Formatos Recomendados

- **Audio comprimido/sin comprimir**: `.wav`, `.mp3`, `.flac`, `.m4a`, `.ogg`
- **Secuencias y partituras**: `.mid`, `.midi`, `.musicxml`
- **Notas y metadatos**: `.json`, `.txt`, `.md` (mismo nombre que la canción con `.md` para añadir notas de intención o contexto).
